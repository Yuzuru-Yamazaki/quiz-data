# Quiz Data

JSON quiz files for use with [quiz-site](https://quiz-site-olive.vercel.app).

## Usage

Copy any raw GitHub URL and paste it into the quiz site.

## Schema

```json
[
  {
    "id": 1,
    "question": "What is the capital of France?",
    "options": ["London", "Berlin", "Paris", "Madrid"],
    "correctAnswer": "Paris"
  }
]
```
